#  Algorithm Comparison Visualizer (DAA Project)

##  Overview
This project is a C++ based Algorithm Visualizer that demonstrates and compares different Design and Analysis of Algorithms concepts.

It includes:
- Sorting Algorithms (Merge Sort, Quick Sort)
- Knapsack Problem (Fractional & 0/1)
- Graph Algorithms (Dijkstra & Prim)



##  Features
- Step-by-step visualization of sorting
- Time comparison of algorithms
- Greedy vs Dynamic Programming comparison
- Graph algorithm execution and result display
- Clean console-based UI



## Algorithms Implemented

### 1. Sorting
- Merge Sort
- Quick Sort

### 2. Knapsack
- Fractional Knapsack (Greedy)
- 0/1 Knapsack (Dynamic Programming)

### 3. Graph
- Dijkstra Algorithm (Shortest Path)
- Prim’s Algorithm (Minimum Spanning Tree)



## Technologies Used
- C++
- STL (Vector, Queue, Priority Queue)
- Chrono (for time measurement)
